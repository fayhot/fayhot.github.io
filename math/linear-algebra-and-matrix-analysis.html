Vector Spaces and Subspaces

  1. A real vector space is a set of “vectors” together with rules for vector addition and for multiplication by real numbers.
  2. A subspace of a vector space is a set of vectors (including 0) that satisfies two requirements: If v\mathbf vv and w\mathbf ww are vectors in the subspace and c is any scalar, then
  1. v\mathbf vv + w\mathbf ww is in the subspace.
  2. cvc\mathbf vcv is in the subspace.
  1. A subspace containing v and w must contain all linear combinations cvc\mathbf vcv + dwd\mathbf wdw.
  2. The column space consists of all linear combinations of the columns .The combinations are all possible vectors AxA\mathbf xAx. They fill the column space C(A)\mathbf C(A)C(A).

The system Ax=bA\mathbf x = \mathbf bAx=b is solvable if and only if b\mathbf bb is in the column space of A.

  1. The nullspace N(A)N(A)N(A) in RnR^nRn contains all solutions x\mathbf xx to Ax=0A\mathbf x = 0Ax=0. This include x=0\mathbf x = 0x=0
  2. Elimination (from AAA to UUU to RRR) does not change the nullspace: N(A)=N(U)=N(R)N(A) = N(U)= N(R)N(A)=N(U)=N(R).
  3. Suppose Ax=0A\mathbf x = 0Ax=0 has more unknowns than equations (n > m, more columns than rows). There must be at least one free column. Then Ax = 0 has nonzero solutions.

A short wide matrix (n>m)(n > m)(n>m) always has nonzero vectors in its nullspace. There must be at least n−mn - mn−m free variables, since the number of pivots cannot exceed m. (The matrix only has m rows, and a row never has two pivots.) Of course a row might have no pivot­ which means an extra free variable. But here is the point: When there is a free variable, it can be set to 1. Then the equation Ax=0A\mathbf x = 0Ax=0 has at least a line of nonzero solutions.

  1. The nullspace is a subspace. Its “dimension” is the number of free variables. This central idea-the dimension of a subspace
  2. The rank of AAA is the number of pivots. This number is rrr. Every ''free column" is a combination of earlier pivot columns. It is the special solutions s\mathbf ss that tell us those combinations.
