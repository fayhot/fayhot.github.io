Image Proccessing

@(Point Operators)[pixel||color|compositing]

@(Pixel)

Two commonly used point processes are multiplication and addition with a constant
g(x) = af (x) + b \tag{1.0.1}
The parameters a>0a > 0a>0 and bbb are often called the gain and bias parameters; sometimes these parameters are said to control contrast and brightness.
The bias and gain parameters can also be spatially varying g(x)=a(x)f(x)+b(x)g(x) = a(x)f(x) + b(x)g(x)=a(x)f(x)+b(x)

Multiplicative gain (both global and spatially varying) is a linear operation, since it obeys the superposition principle h(f0+f1)=h(f0)+h(f1)h(f_0 + f_1) = h(f_0) + h(f_1)h(f0​+f1​)=h(f0​)+h(f1​)

Another commonly used dyadic (two-input) operator is the linear blend operator
g(x) = (1-\alpha)f_0(x) + \alpha f_1(x) \tag{1.0.2}

@(Compositing and matting)

Compositing equation C=(1−α)B+αFC = (1-\alpha)B + \alpha FC=(1−α)B+αF.

This operator attenuates the influence of the background image B by a factor (1 − α) and then adds in the color (and opacity) values corresponding to the foreground layer F

@(Linear Filtering)[]

The most commonly used type of neighborhood operator is a linear filter, in which an output pixel’s value is determined as a weighted sum of input pixel values

g(i,j)=∑k,lf(i+k,j+l)h(k,l)(1.0.1)\displaystyle g(i,j) = \sum_{k,l}f(i+k, j+l)h(k,l) \tag{1.0.1} g(i,j)=k,l∑​f(i+k,j+l)h(k,l)(1.0.1)

The entries in the weight kernel or mask h(k, l) are often called the filter coefficients. The above correlation operator can be more compactly notated as g=f⊗hg = f\otimes hg=f⊗h

A common variant on this formula is

g(i,j)=∑k,lf(i−k,j−l)h(k,l)=∑k,lf(k,l)h(i−k,j−l)(1.0.2)\displaystyle g(i,j) = \sum_{k,l}f(i-k,j-l)h(k,l) = \sum_{k,l}f(k,l)h(i-k, j-l) \tag{1.0.2} g(i,j)=k,l∑​f(i−k,j−l)h(k,l)=k,l∑​f(k,l)h(i−k,j−l)(1.0.2)

where the sign of the offsets in f has been reversed. This is called the convolution operator, g=f∗hg = f*hg=f∗h
