@(Linear Filtering)[]

The most commonly used type of neighborhood operator is a linear filter, in which an output pixel’s value is determined as a weighted sum of input pixel values

g(i,j)=∑k,lf(i+k,j+l)h(k,l)(1.0.1)\displaystyle g(i,j) = \sum_{k,l}f(i+k, j+l)h(k,l) \tag{1.0.1} g(i,j)=k,l∑​f(i+k,j+l)h(k,l)(1.0.1)

The entries in the weight kernel or mask h(k, l) are often called the filter coefficients. The above correlation operator can be more compactly notated as g=f⊗hg = f\otimes hg=f⊗h

A common variant on this formula is

g(i,j)=∑k,lf(i−k,j−l)h(k,l)=∑k,lf(k,l)h(i−k,j−l)(1.0.2)\displaystyle g(i,j) = \sum_{k,l}f(i-k,j-l)h(k,l) = \sum_{k,l}f(k,l)h(i-k, j-l) \tag{1.0.2} g(i,j)=k,l∑​f(i−k,j−l)h(k,l)=k,l∑​f(k,l)h(i−k,j−l)(1.0.2)

where the sign of the offsets in f has been reversed. This is called the convolution operator, g=f∗hg = f*hg=f∗h

@(Separable filtering)

The process of performing a convolution requires K2K^2K2 (multiply-add) operations per pixel, where KKK is the size (width or height) of the convolution kernel. In many cases, this operation can be significantly sped up by first performing a one-dimensional horizontal convolution followed by a one-dimensional vertical convolution (which requires a total of 2K operations per pixel). A convolution kernel for which this is possible is said to be separable.
It is easy to show that the two-dimensional kernel K corresponding to successive con- volution with a horizontal kernel h and a vertical kernel v is the outer product of the two kernels
K = vh^T \tag{1.0.3}

How can we tell if a given kernel KKK is indeed separable? This can often be done by inspection or by looking at the analytic form of the kernel. A more direct method is to treat the 2D kernel as a 2D matrix K and to take its singular value decomposition (SVD)

K=∑iσiuiviT(1.0.4)\displaystyle K = \sum_i\sigma_iu_iv_i^T \tag{1.0.4} K=i∑​σi​ui​viT​(1.0.4)

If only the first singular value σ0\sigma_0σ0​ is non-zero, the kernel is separable and σ0u0\sqrt{\sigma_0}u_0σ0​​u0​ and σ0v0T\sqrt{\sigma_0}v_0^Tσ0​​v0T​ provide the vertical and horizontal kernels (Perona 1995). For example, the Laplacian of Gaussian kernelcan be implemented as the sum of two separable filters.
